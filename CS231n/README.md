# CS231n

《CS231n: Deep Learning for Computer Vision》的课程笔记，包含了三个 Assignments，参考 [官网](https://cs231n.stanford.edu/2025/index.html)、[B 站课程视频](https://www.bilibili.com/video/BV1b1agz5ERC/)

测试环境：Google Colab，参考 [CS231N Google Colab Assignment Workflow Tutorial](https://www.youtube.com/watch?v=DsGd2e9JNH4)

需要注意，对于 Assignment 3（`CLIP_DINO.ipynb` 除外），需在 Colab 右下角点击 Python 3 选项，选择 “更改运行时类型” 将 “运行时版本” 修改为 “2025.07”，而 `CLIP_DINO.ipynb` 依然在 “运行时版本” 中选择 “最新”。
